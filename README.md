# AWS EC2 Web Server Deployment

## Project Overview
This project demonstrates deployment of a static website using AWS EC2 Ubuntu instance.

## AWS Services Used
- EC2
- Security Groups

## Operating System
- Ubuntu Linux

## Web Server
- Apache2

## Steps Performed
1. Launched Ubuntu EC2 instance
2. Configured Security Groups for SSH and HTTP
3. Connected using SSH using .pem key
4. Installed Apache Web Server
5. Hosted static website using EC2 public IP

## Ports Used
- 22 (SSH)
- 80 (HTTP)

## Learning Outcome
- Learned EC2 setup
- Understood Security Groups
- Practiced Linux commands
- Gained hands-on AWS experience
